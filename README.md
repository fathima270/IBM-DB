# IBM Sales Report Dashboard 📊

## 📌 Project Overview

The **IBM Sales Report Dashboard** is an interactive data visualization project designed to analyze employee and sales-related metrics within an organization. Built using modern business intelligence tools, this dashboard provides insights into workforce distribution, income patterns, employee satisfaction, and attrition trends.

The goal of this project is to enable **data-driven decision-making** by transforming raw HR and sales data into meaningful visual insights.

---

## 🚀 Key Features

* **KPI Summary Cards**

  * Total Monthly Income: **9M**
  * Total Employees: **1383**
  * Attrition Count: **226**
  * Sum of Job Levels: **2774**

* **Department-wise Analysis**

  * Monthly income distribution across departments
  * Employee count breakdown (Donut Chart)
  * Hourly rate comparison

* **Employee Insights**

  * Job satisfaction by job role (Pie Chart)
  * Attrition tracking
  * Gender-based filtering
  * Age range filtering (interactive slider)

* **Interactive Filters**

  * Department selection
  * Gender filter (Male/Female)
  * Age range slicer

---

## 📊 Visualizations Used

* Bar Chart → Monthly income by department
* Donut Chart → Employee count distribution
* Pie Chart → Job satisfaction by role
* Line Chart → Hourly rate trends
* KPI Cards → High-level metrics
* Slicers → Dynamic filtering (Department, Gender, Age)
